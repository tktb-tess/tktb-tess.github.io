## 母音

ヴェッセンズラン語の母音字には **tense vowel** と **lax vowel** の2通りの読み方がある。概ね長母音と短母音のような関係である。

主な母音字の *tense, lax* の音は以下の通り。

<div class="[&_:where(tbody_td:not(:first-child))]:font-ipa">

| 母音字 | tense | lax |
| ------ | ----- | --- |
| a      | /ɑː/  | /ä/ |
| e      | /ɛː/  | /ɛ/ |
| ê      | /eː/  | /e/ |
| i, y   | /iː/  | /ɪ/ |
| o      | /oː/  | /ɔ/ |
| ô      | /uː/  | /ʊ/ |
| œ      | /øː/  | /œ/ |
| u      | /yː/  | /ʏ/ |

</div>

基本原則は、

- 後ろに子音が続かない場合、強勢の有無に拘わらず *tense*
- 後ろに子音が続き、強勢がある場合、
  - 子音字が1文字のみの場合 *tense*
  - 2文字以上の場合 *lax*
- 後ろに子音が続き、強勢がない場合、大体 *lax*

である。

e.g.  
fnut <span class="font-ipa">/fnyːtʰ/</span> vs. hund <span class="font-ipa">/ɦʏn/</span>

ただし、前置詞や代名詞など使用頻度の高い機能語はこの限りではない。また、 ‹gh› 等の digraph が後ろに続く場合、必ずしも *lax* とは限らない。

e.g.  
iç <span class="font-ipa">/ɪɕ/</span>, hvat <span class="font-ipa">/fätʰ/</span>

*tense* 母音の対応する二重音字も存在する。

<div class="[&_:where(tbody_td:not(:first-child))]:font-ipa">

| 綴り   | 発音 |
| ------ | ---- |
| aa     | /ɑː/ |
| ae     | /ɛː/ |
| ee     | /eː/ |
| ie, ye | /iː/ |
| ao     | /oː/ |
| oo     | /uː/ |
| œu     | /øː/ |
| ue     | /yː/ |

</div>

二重母音は以下の通りに綴る。

<div class="[&_:where(tbody_td:not(:first-child))]:font-ipa">

| 綴り | 発音  |
| ---- | ----- |
| aw   | /ɑʊ/  |
| aj   | /aɪ/  |
| ej   | /ɛɪ/  |
| uj   | /œʏ/  |
| ia   | /jɑː/ |
| ew   | /joː/ |

</div>
